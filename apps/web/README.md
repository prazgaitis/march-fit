# March Fit Web App

Next.js 15 frontend for March Fit.

## Development

From the repo root:

```bash
pnpm dev
```

The web app runs at http://localhost:3001.

## Local Convex

```bash
pnpm dev:local
```

## Tests

```bash
pnpm -F web test
```

Project-wide documentation is in the repo root `README.md` and `docs/`.
